Step-by-Step Lockbox Installation


When your lockbox arrives in the mail, it will already have a code programmed.  Change the numbers on the lockbox to match the code assigned to the lockbox.


Once you have the code entered, you will be able to open the lockbox. To do this push down on this lever to release the lock and open.


Next step, release the top shackle. To do this you will slide this inside lever left and the shackle will open.


This allows you to securely attach it to your door. Once attached, give the box a light tug to make sure it is on securely.


Now take your key and put it inside th compartment in the box. Make sure it is inside the compartment, otherwise the key could cause the box to get jammed.


Once done, make sure and turn the tumblers and test that the box is locked.


Your lockbox is now installed and ready to be used.
