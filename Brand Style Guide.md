# Access Realty Color Guide

This document outlines the color selections used throughout the Access Realty application, based on the brand style guide.

---

## Brand Colors

### Primary Navy
The main brand color used for the sidebar, primary buttons, and key UI elements.

| Format | Value |
|--------|-------|
| Hex    | `#284b70` |
| RGB    | `rgb(40, 75, 112)` |

**Usage:** Sidebar background, primary action buttons (Next), selected sidebar items text, card title text.

---

### Navy Light (Hover/Header)
A lighter navy used for hover states and secondary backgrounds.

| Format | Value |
|--------|-------|
| Hex    | `#3a6690` |
| RGB    | `rgb(58, 102, 144)` |

**Usage:** Card header background, sidebar hover states, selected option buttons.

---

### Tan Dark (Gold Accent)
The primary gold/tan accent color for highlighting active elements.

| Format | Value |
|--------|-------|
| Hex    | `#d6b283` |
| RGB    | `rgb(214, 178, 131)` |

**Usage:** Active sidebar menu item background, progress bar fill, avatar ring on active user.

---

### Tan Light
A softer tan used for subtle backgrounds and hover states.

| Format | Value |
|--------|-------|
| Hex    | `#d1b99b` |
| RGB    | `rgb(209, 185, 155)` |

**Usage:** Option button hover background (at 40% opacity), progress bar track, subtle borders.

---

### Cream (Background)
The warm off-white used for page backgrounds.

| Format | Value |
|--------|-------|
| Hex    | `#f8f4ef` |
| RGB    | `rgb(248, 244, 239)` |

**Usage:** Main content area background, form section backgrounds.

## Typography

### Logo Fonts

| Element | Font Family | Style |
|---------|-------------|-------|
| "Access" (logo) | Times New Roman MT | Italic |
| "Realty" (logo) | Be Vietnam | Bold |

### Application Fonts

| Purpose | Font Family | Weight |
|---------|-------------|--------|
| Headings | Verdana | Regular/Bold |
| Body text | Arial | Regular |

### Font Pairing Notes

- **Logo:** The combination of Times New Roman MT Italic ("Access") with Be Vietnam Bold ("Realty") creates a classic-meets-modern look that conveys both trustworthiness and approachability
- **Times New Roman MT Italic** - Serif font providing elegance and established credibility
- **Be Vietnam Bold** - Clean sans-serif that adds contemporary professionalism

---

## Notes

- Always ensure sufficient contrast between text and background colors
- Use the tan/gold colors sparingly as accents to maintain visual hierarchy
- The navy colors should anchor navigation and primary actions
- The cream background creates warmth while allowing white cards to stand out
