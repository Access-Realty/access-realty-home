================================================================================
                     ACCESS REALTY MARKETING SITE ARCHITECTURE
                              access.realty
================================================================================

This document describes the page structure, navigation pathways, and key
elements for the Access Realty marketing website.

================================================================================
                              ROUTE GROUPS
================================================================================

The site uses Next.js route groups to apply different layouts:

  /(marketing)     - Main marketing pages (Header + Footer)
  /(direct-list)   - DirectList landing page (DirectListHeader + DirectListFooter)
  /(staff)         - Staff/team pages (StaffHeader + Footer)

================================================================================
                           EXTERNAL LINKS
================================================================================

All authentication flows redirect to the main application:

  Sign In  →  https://access.realty/app
  Sign Up  →  https://access.realty/app/signup

With tracking:
  Get Started (from home)  →  access.realty/app/signup?source=home&address=...
  Plan selection           →  access.realty/app/signup?plan=direct-list

================================================================================
                          LOGO STYLING PATTERN
================================================================================

Brand logos use consistent font styling:

  "Access Realty" Logo:
    - "Access" → Times New Roman, italic
    - "Realty" → Be Vietnam Pro, bold weight 700

  "DirectList" Logo:
    - "Direct" → Times New Roman, italic
    - "List"   → Be Vietnam Pro, bold weight 700

  "Full Service" (when styled):
    - "Full"    → Times New Roman, italic
    - "Service" → Be Vietnam Pro, bold weight 700

================================================================================
                            COLOR SCHEME
================================================================================

  Primary:    #284b70  (Navy)      - Headers, buttons, accents
  Secondary:  #d6b283  (Tan/Gold)  - CTAs, highlights, badges
  Background: #f8f4ef  (Cream)     - Page backgrounds

================================================================================
                           FILE STRUCTURE
================================================================================

  app/
    (marketing)/
      layout.tsx          ← Uses Header + Footer
      page.tsx            ← Homepage
      services/page.tsx   ← Pricing comparison
      how-it-works/page.tsx
      privacy/page.tsx
      terms/page.tsx

    (direct-list)/
      direct-list/
        page.tsx          ← DirectList landing (different header/footer)

    (staff)/
      layout.tsx          ← Uses StaffHeader + Footer
      staff/
        page.tsx          ← Team directory
        [slug]/page.tsx   ← Individual profiles

  components/
    Header.tsx            ← Main header (Solutions, How It Works, Sign In)
    Footer.tsx            ← Main footer (full Quick Links)
    StaffHeader.tsx       ← Simplified header (logo + Sign In only)
    direct-list/
      DirectListHeader.tsx ← DirectList header (How It Works, Compare Plans)
      DirectListFooter.tsx ← DirectList footer (DirectList-focused links)
